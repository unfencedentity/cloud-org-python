# cloud-org-python

Enterprise Python engineering foundation for automation, APIs, cloud operations, DevOps tooling and AI systems.

This repository is part of the cloud-org ecosystem.

## Mission

This is not a Python syntax exercise repository.

The goal is to learn Python through real engineering problems, production-style structure, clean code, testing, logging, automation, API clients, CLI tools and cloud scenarios.

## Learning Philosophy

We learn using baby steps.

Each concept must be deeply understood before moving forward.

The objective is not to memorize Python syntax.

The objective is to build a strong engineering foundation for Cloud, DevOps and Artificial Intelligence.

## Repository Goals

This repository will build foundations for:

- clean Python applications
- automation scripts
- REST API clients
- Azure automation
- GitHub automation
- CLI tools
- JSON processing
- file handling
- logging
- configuration management
- testing
- object-oriented programming
- asynchronous programming
- FastAPI fundamentals
- reusable Python packages
- enterprise project structure

## Engineering Standard

All code, documentation, commits and naming conventions are written in English.
